# HỆ THỐNG ĐẶT PHÒNG KHÁCH SẠN TRỰC TUYẾN
## Giới thiệu về dự án
Hệ thống đặt phòng khách sạn trực tuyến là một ứng dụng web cho phép người dùng tìm kiếm, đặt phòng và quản lý các đặt phòng khách sạn một cách dễ dàng và tiện lợi. Dự án này nhằm mục đích cung cấp một nền tảng trực tuyến để kết nối khách hàng với các khách sạn, giúp họ tiết kiệm thời gian và công sức trong việc đặt phòng.
## Tính năng chính
1. Tìm kiếm khách sạn: Người dùng có thể tìm kiếm khách sạn dựa trên các tiêu chí như địa điểm, ngày nhận phòng, ngày trả phòng, số lượng phòng và loại phòng.
2. Đặt phòng: Người dùng có thể chọn phòng phù hợp và tiến hành đặt phòng trực tuyến thông qua hệ thống.
3. Quản lý đặt phòng: Người dùng có thể xem, sửa đổi hoặc hủy bỏ các đặt phòng đã thực hiện.
4. Đánh giá và nhận xét: Người dùng có thể đánh giá và để lại nhận xét về các khách sạn mà họ đã ở.
5. Quản lý khách sạn: Quản trị viên có thể thêm, sửa đổi hoặc xóa thông tin khách sạn và quản lý các đặt phòng.
## Công nghệ sử dụng
- Frontend: Nextjs, Shadcn UI
- Backend: Java Spring Boot (RESTful API)
- Triển khai: Docker và Kubernetes
- Java version: 21
- Cơ sở dữ liệu: PostgreSQL
- Công cụ quản lý phiên bản: Git và GitHub
